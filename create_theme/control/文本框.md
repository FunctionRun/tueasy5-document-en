# 文本框\_Text ![](/assets/text.png)
## User Configuration  
Users can through the bottom of the edit mode change the background color of the text box, color, font size.  
![](/assets/textUser.jpg)

## Code Model 

### API

```
{
	'type': 'Text',
	'css': {
		'boxSizing': 'border-box',
		'width': '100%',
		'height': '100%',
		'backgroundColor': 'rgba(0,0,0,0)',
		'color': '#333',
		'fontFamily': 'Hiragino Sans GB',
		'fontSize': 20,
		'lineHeight': '36px',
		'borderWidth': 2,
		'borderRadius': 5,
		'borderColor': '#BCBCBC',
		'borderStyle': 'solid',
		'textAlign': 'left',
		'paddingLeft': '10px'
	},
	'animation': {
		'show': true,
		'animationDuration': 0,
		'animationInterval': 0
	},
	'text': '',
	'postfix': ''
}
```
